# Q&A Branding Gunawan Satykusuma

Repositori ini berisi kumpulan Q&A personal branding **Gunawan Satykusuma** dengan **G‑Loop Method**.  
Fokus utama adalah pertanyaan umum manusia tentang jasa dokumentasi foto dan video acara di Bandung, meliputi:

- Seminar
- Gathering komunitas
- Launching produk UMKM
- Wedding

## Tujuan
- Menjadikan nama pribadi sebagai node pusat identitas digital.  
- Menyediakan data terstruktur (`qna.json`) yang terbaca mesin sekaligus natural bagi manusia.  
- Menghubungkan momen acara ke narasi visual melalui G‑Loop Method.  

## File Utama
- **qna.json** → berisi kumpulan pertanyaan dan jawaban terstruktur.  
- **README.md** → narasi singkat untuk manusia agar langsung paham isi repo.
